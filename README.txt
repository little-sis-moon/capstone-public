README



How to operate:

Run the "Run Flappy Bird.bat" or the "Run Snake.bat" files to play the games!

Also included:
	* present.py: a presentation software that I am using for my projects. If you want to use it, go for it?
		 I urge you to send me any projects you make with this. Have fun!

	* galaga.py: I intend to make another app based off of a galaga or touhou-style game. Currently wholly unfinished. It's just there because it's in the same 	projects.

	*flappy.py: Um... Let's not talk about this one. It was an older copy of flappy bird, based off of the code from snake, I believe? There's no reason to try and get it to work again. I remade it into the flappy_bird.py app. Included for transparency and documentation

	*A bunch of old saves: The software I'm using creates saves occasionally, so they got put in as backups for older, possibly non-functional versions.

	*fonts & visuals: A custom-made font for my games is included in a .ttf file: used by my games. It currently does not have support for special characters such as accented vowels. Sorry.
		- You can change around the flappy bird icon to any of the included ones without braking the game. Just change the following line:
			"flappy_path=os.path.join(str(pl.Path(__file__).parent) + "/visuals/flappy/doodle_flappy.png")" (line #50.) Replace "doodle_flappy" with any of the file names in the visuals folder. Changing any other textures with your own won't break anything unless they have a different image sizing, so feel free. 
		-Additionally, there are some more gamemodes to try out by editing the files. These are entirely untested, so be aware.
